# RevokeBotMessage 插件（撤回机器人消息）

## 功能

- 触发词：`撤回`
- 权限：仅 `main_config.toml` 中 `XYBot.admins` 全局管理员可使用
- 用法1：**引用机器人发出的消息**，发送 `撤回`，撤回被引用消息
- 用法2：不引用消息，直接发送 `撤回`，撤回当前会话中机器人最后一条消息
- 若撤回失败/过期：返回提示 `撤回过期 时间过了`

## 说明

撤回接口需要 `ClientMsgId/CreateTime/NewMsgId` 三元组。插件会在运行时“包装”框架的发送方法，记录机器人发出去的回执，因此：

- 插件启用后发送的消息都可撤回（文本/卡片/图片/文件等，只要发送接口返回了回执）
- 插件启用前发送的消息，可能无法撤回（缺少回执）

## 配置

配置文件：`plugins/RevokeBotMessage/config.toml`

- `enable`：是否启用
- `trigger`：触发词，默认 `撤回`
- `max_age_seconds`：超过该秒数的消息直接提示“撤回过期”，默认 `120`
