# AllBot 用户手册

## 1. 快速开始

### 1.1 登录管理后台

AllBot 提供了一个强大的管理后台，您可以通过它来管理和监控机器人：

1. 在浏览器中打开 `http://[服务器IP]:9090`（默认端口为 9090）
2. 输入用户名和密码登录（账号信息以 `main_config.toml` 的 `[Admin]` 配置为准）
3. 登录后进入控制面板

### 1.2 了解管理后台界面

管理后台包含以下主要功能模块：

- **控制台**：系统状态与运行信息
- **插件管理**：启用、禁用或配置插件
- **适配器管理**：管理多平台接入
- **Web 对话**：直接在后台模拟对话流程
- **文件管理**：查看与管理文件资源
- **联系人/群管理**：查看联系人与群相关信息
- **通知管理**：配置系统通知

## 2. Web 对话功能

### 2.1 功能概述

Web 对话功能允许您通过管理后台界面直接与机器人进行对话，模拟真实的消息处理流程。消息会通过适配器机制发送到消息队列，由插件系统处理，然后回复通过回复队列返回到 Web 界面。

### 2.2 使用方法

1. 在管理后台左侧菜单点击 **Web 对话**
2. 点击 **新建会话** 创建会话
3. 输入消息并发送
4. 等待机器人回复

### 2.3 注意事项

- 确保 Web 适配器已启用
- 消息回复超时默认为 30 秒
- 会话信息存储在内存中，重启服务后会清空

## 3. 多平台适配器

### 3.1 适配器概述

AllBot 支持多种平台接入，包括：

- **Web 适配器**：用于管理后台 Web 对话
- **QQ 适配器**：接入 QQ 消息
- **Telegram 适配器**：接入 Telegram 消息

### 3.2 管理适配器

1. 进入 **适配器管理** 页面
2. 查看适配器状态与配置
3. 启用或禁用适配器（修改后需重启服务生效）

## 4. 插件使用指南

### 4.1 插件管理

1. 进入 **插件管理** 页面
2. 查看插件列表及状态
3. 点击启用/禁用并按提示重启服务

### 4.2 插件配置

1. 进入插件详情页面
2. 修改配置并保存
3. 重启服务使配置生效

## 5. 常见问题

### 5.1 Web 对话无法发送消息

- 检查 Web 适配器是否已启用
- 确认 Redis 服务可用
- 查看系统日志定位错误原因

### 5.2 适配器无法启动

- 检查适配器配置文件是否正确
- 确保外部依赖（如 Redis、RabbitMQ）可用
- 查看适配器日志排查问题
