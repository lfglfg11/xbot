## 📖 使用方法

### 查看插件详情

要查看某个插件的详细信息，可以访问对应的 README 文件：

```bash
# 查看插件文档
cat plugins/[插件名]/README.md
```

### 启用/禁用插件

在 `main_config.toml` 中配置：

```toml
[XYBot]
disabled-plugins = ["插件名1", "插件名2"]  # 填入要禁用的插件名称
```

### 插件配置

每个插件通常都有自己的 `config.toml` 配置文件，位于：

```
plugins/[插件名]/config.toml
```
